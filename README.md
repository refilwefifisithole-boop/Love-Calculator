 ❤️ AstroMatch ❤️

 About the Project

AstroMatch is a desktop Java application that calculates relationship compatibility using a combination of astrology, zodiac signs, birth charts, and numerology. Rather than generating a random percentage, the application analyzes multiple compatibility factors to provide a detailed compatibility score and personalized relationship insights.

This project was created to strengthen my Java programming skills while building a unique portfolio application that demonstrates object-oriented programming, problem-solving, and software development best practices.

This application is intended for entertainment purposes. Astrology-based compatibility is not scientifically validated.

 Features

Current Features

* Calculate Zodiac (Sun) signs from birth dates
* Determine zodiac elements (Fire, Earth, Air, Water)
* Analyze zodiac modalities (Cardinal, Fixed, Mutable)
* Calculate Life Path Numbers using numerology
* Generate an overall compatibility score
* Display compatibility explanations instead of just a percentage
* User-friendly desktop interface

How It Works

The application calculates compatibility by combining several independent scores, including:

* Zodiac Sign Compatibility
* Zodiac Element Compatibility
* Zodiac Modality Compatibility
* Life Path Number Compatibility
* Birth Chart Compatibility
* Planetary Alignment 

These scores are combined to generate an overall compatibility percentage along with a detailed explanation of the results.
